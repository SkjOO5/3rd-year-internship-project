# 3rd Year Internship Project

This repository contains the codebase for my 3rd-year internship project. 
Link - https://skjoo5.github.io/3rd-year-internship-project/

## Project Structure

The project consists of the following key pages and assets:
* **[index.html]** - The main landing page.
* **[intro.html]** - Introduction to the project/system.
* **[about.html]** - Information about the project scope and background.
* **[projects.html]** - Displays the various projects undertaken during the internship.
* **[portfolio.html]** - Internship portfolio and accomplishments.
* **[form.html]** - Interactive forms and data collection pages.
* **[contact.html]** - Contact information and feedback.
* **Landscape.png** - Visual asset for the website.
